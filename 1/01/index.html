一、流程介紹

  • Host app：使用者透過哪個app（本機照片、檔案或連結）分享到我的app，透過的這個app稱為Host app。

  • App extension：由Extension負責接收從Host app傳進來的檔案。

  • Containing app：主要開發的Project app，負責將資料如何對外呈現。(股市爆料同學會)

  • Shared resources：Extension與Containing app透過蘋果提供的App Groups，允許開發者能夠讓Containing app存取Extension所接收到的檔案。

    • UserDefaults
    • FileManager
    • CoreData

二、實作

1. File → New → Target → ShareExtension

新增ShareExtension Target之後自動產生3個檔案

  • info.plist：設定可以分享至APP的內容格式、ShareExtension入口的ViewController等。
  • MainInterface.storyboard：點擊分享打開ShareExtension的入口
  • ShareViewController: SLComposeServiceViewController：預設的分享彈窗

2. 設定 App groups

  • 一般的情況下，iOS App 只能存取自己資料夾下的東西，無法存取其它 App 的內容。
  • App Group 讓不同的 App 或是 App & App extension 之間可以共享資料。
  • 利用 UserDefaults 共享資料，輸入的 container 名字要與 UserDefaults 的 suitName 相同。

3. 在Info.plist設定APP允許接受什麼類型的內容

NSExtension ➔ NSExtensionAttributes 改為Dictionary Type ➔ 新增下列欲選取的key

Key 說明
NSExtensionActivationSupportsAttachmentsWithMaxCount 附件最多數量限制。附件類型包括File、Image、movie，可以單一、可以複選，但總數量不得超過指定數量
NSExtensionActivationSupportsAttachmentsWithMinCount 附件最少數量限制。預設至少選擇1個附件分享時才會顯示app icon
NSExtensionActivationSupportsFileWithMaxCount 文件最多數量限制。泛指除Image/Movie之外的附件。
NSExtensionActivationSupportsImageWithMaxCount 圖片最多數量限制。
NSExtensionActivationSupportsMovieWithMaxCount 影片最多數量限制。
NSExtensionActivationSupportsText 是否支持文本類型。
NSExtensionActivationSupportsWebURLWithMaxCount web連結最多數量限制。
NSExtensionActivationSupportsWebPageWithMaxCount web頁面最多數量限制。

4. 透過解析extensionContext得到分享內容

NSExtensionContext

  • inputItems: [NSExtensionItem]：分享一種類型為一個item。
  • completeRequest(returningItems: [Any]?, completionHandler: ((Bool) -> Void)?)：完成分享請求。
  • cancelRequest(withError: Error)：取消分享請求。

NSExtensionItem

  • attributedTitle: 標題
  • attributedContentText: 内容
  • attachments: 附件陣列，包含圖片、影片、連結等資源，封裝在NSItemProvider類中
  • userInfo: [AnyHashable: Any]：是一個字典，NSExtensionItem中的屬性都會在裡面
    Key 說明
    NSExtensionItemAttributedTitleKey 標題
    NSExtensionItemAttributedContentTextKey 內容
    NSExtensionItemAttachmentsKey 附件

NSItemProvider常用方法

  1. 判斷是否有指定類型的資源： hasItemConformingToTypeIdentifier(typeIdentifier: String) -> Bool
  2. 加載指定類型的資源： loadItem(forTypeIdentifier: String, options: [AnyHashable : Any]?, completionHandler: NSItemProvider.CompletionHandler?)
  3. 加載資源的預覽圖片： loadPreviewImage(options: [AnyHashable : Any]?, completionHandler: NSItemProvider.CompletionHandler?)

- 啟動Project App的方法

[注意] ShareExtension無法使用extensionContext.open(URL: URL, completionHandler: ((Bool) -> Void)?) 的方式開啟Project App

必須使用方法openURL(_ url: URL)如下： (可以通過審核！) 

private let appURL: String = "forum://shareExtension"

private func openMainApp() {
    extensionContext?.completeRequest(returningItems: nil, completionHandler: { [weak self] _ in
        guard let self = self,
              let url = URL(string: self.appURL) else { return }
        _ = self.openURL(url)
    })
}

/// 開啟app連結
@objc private func openURL(_ url: URL) -> Bool {
    var responder: UIResponder? = self
    while responder != nil {
        if let application = responder as? UIApplication {
            return application.perform(#selector(openURL(_:)), with: url) != nil
        }
        responder = responder?.next
    }
    return false
}

在Project APP的Info.plist設定URL Scheme (ex: forum)

並在Project APP的AppDelegate設定可以接收到shareExtension傳進來的連結(ex:“forum://shareExtension”)

func application(_ app: UIApplication, open url: URL, options: [UIApplication.OpenURLOptionsKey : Any] = [:]) -> Bool {
    if let components = NSURLComponents(url: url, resolvingAgainstBaseURL: true),
    components.host == "shareExtension" {
        return true
    }
    return false
}

    /// 處理分享內容
    private func handleIncomingContext(type: CreateType) {
        isHandlingIncomingContext = true
        guard let extensionItem = context?.inputItems.first as? NSExtensionItem,
              let attachments = extensionItem.attachments else {
            handleIncomingError()
            return
        }
        shareData.imageData = []
        for (index, itemProvider) in attachments.enumerated() {
            if itemProvider.hasItemConformingToTypeIdentifier(typeText) {
                handleIncomingText(type: type, itemProvider: itemProvider)
            } else if itemProvider.hasItemConformingToTypeIdentifier(typeURL) {
                handleIncomingURL(type: type, itemProvider: itemProvider)
            } else if itemProvider.hasItemConformingToTypeIdentifier(typeImage){
                handleIncomingImage(type: type, itemProvider: itemProvider, isLast: index == attachments.count-1)
            } else {
                handleIncomingError()
            }
        }
        
    }
    
    /// 處理錯誤，顯示彈窗
    private func handleIncomingError() {
        let alert = UIAlertController(title: NSLocalizedString("分享失敗", comment: ""), message: NSLocalizedString("無法辨識您所分享的內容", comment: ""), preferredStyle: .alert)
        
        let action = UIAlertAction(title: NSLocalizedString("確定", comment: ""), style: .cancel) { [weak self] _ in
            self?.dismiss(animated: true, completion: nil)
            self?.context?.cancelRequest(withError: ShareError(description: "Cancel"))
        }
        
        alert.addAction(action)
        present(alert, animated: true, completion: nil)
    }
    
    /// 處理照片
    private func handleIncomingImage(type: CreateType, itemProvider: NSItemProvider, isLast: Bool) {
        itemProvider.loadItem(forTypeIdentifier: typeImage, options: nil) { [weak self] (item, error) in
            guard let self = self else { return }
            if let error = error {
                if isLast {
                    self.isHandlingIncomingContext = false
                    self.context?.cancelRequest(withError: error)
                }
                return
            }
            
            // 處理從 project app 點擊圖片分享，又選擇分享至App中發文/提問
            if let image = item as? UIImage,
               let rawData = image.pngData() {
                self.saveImage(type: type, rawData: rawData, isLast: isLast)
                return
            }

            guard let url = item as? URL,
                  let rawData = try? Data(contentsOf: url),
                  let image = UIImage(data: rawData) else {
                if isLast {
                    self.isHandlingIncomingContext = false
                    self.context?.completeRequest(returningItems: nil, completionHandler: nil)
                }
                return
            }
            
            // 從外部分享的照片可能檔案過大造成儲存失敗，先壓縮再儲存
            let compressImage = image.compressImage(targetWidth: 1280)
            let compressImageData = compressImage?.compressImageMid(expectedKb: 200)
            self.saveImage(type: type, rawData: compressData ?? rawData, isLast: isLast)
        }
    }
    
    /// 儲存照片
    private func saveImage(type: CreateType, rawData: Data, isLast: Bool) {
        shareData.imageData.append(rawData)
        
        if isLast {
            saveContent(content: self.shareData)
            openMainApp(type: type)
            isHandlingIncomingContext = false
        }
    }
    
    /// 處理文字
    private func handleIncomingText(type: CreateType, itemProvider: NSItemProvider) {
        itemProvider.loadItem(forTypeIdentifier: typeText, options: nil) { [weak self] (item, error) in
            guard let self = self else { return }
            self.isHandlingIncomingContext = false

            guard error == nil,
                  let text = item as? String else {
                self.handleIncomingError()
                return
            }
            self.shareData.text = text
            self.saveContent(content: self.shareData)
            self.openMainApp(type: type)
        }
    }
    
    /// 處理連結
    private func handleIncomingURL(type: CreateType, itemProvider: NSItemProvider) {
        itemProvider.loadItem(forTypeIdentifier: typeURL, options: nil) { [weak self] (item, error) in
            guard let self = self else { return }
            self.isHandlingIncomingContext = false
            guard error == nil,
                  let url = item as? NSURL,
                  let urlString = url.absoluteString else {
                self.handleIncomingError()
                return
            }
            
            if let url = item as? URL,
               let rawData = try? Data(contentsOf: url),
               let _ = UIImage(data: rawData) {
                // 若url是圖片存成圖片data
                self.shareData.imageData.append(rawData)
                self.saveContent(content: self.shareData)
            } else {
                // 網址不是圖片再存成文字
                self.shareData.text = urlString
                self.saveContent(content: self.shareData)
            }
            self.openMainApp(type: type)
        }
    }

    /// 儲存分享內容至UserDefaults
    private func saveContent(content: ShareData) {
        UserDefaults.syncShareData(content)
    }

    /// 開啟app
    private func openMainApp(type: CreateType) {
        self.context?.completeRequest(returningItems: nil, completionHandler: { [weak self] _ in
            guard let self = self,
                  let url = URL(string: self.appURL + String(type.rawValue)) else { return }
            _ = self.openURL(url)
        })
    }

    /// 開啟app連結
    @objc private func openURL(_ url: URL) -> Bool {
        var responder: UIResponder? = self
        while responder != nil {
            if let application = responder as? UIApplication {
                return application.perform(#selector(openURL(_:)), with: url) != nil
            }
            responder = responder?.next
        }
        return false
    }
    
    // 點擊回饋(peek震動)
    func peekFeedback() {
        let generator = UIImpactFeedbackGenerator(style: .heavy)
        generator.prepare()
        generator.impactOccurred()
    }

/// 分享錯誤物件
struct ShareError: Error {
    var description: String
}


extension UserDefaults {
    
    /// 分享內容存至UserDefaults辨認的字串
    static let sharedIdentifier: String = "group.CMoney.Forum"
    
    /// 儲存至UserDefaults的key
    static let contextSharedKey = "contextSharedKey"
    
    /// 將ShareData存進UserDefaults
    static func syncShareData(_ shareData: ShareData) {
        do {
            // 轉換成功就存進 UserDefaults
            let data = try JSONEncoder().encode(shareData)
            let userDefaults = UserDefaults(suiteName: UserDefaults.sharedIdentifier)
            userDefaults?.set(data, forKey: UserDefaults.contextSharedKey)
        } catch {
            // 轉換失敗就清掉
            let userDefaults = UserDefaults(suiteName: UserDefaults.sharedIdentifier)
            userDefaults?.removeObject(forKey: UserDefaults.contextSharedKey)
        }
    }
    
    /// 取得UserDefaults中的ShareData
    static func getShareData() -> ShareData? {
        if let data = UserDefaults(suiteName: sharedIdentifier)?.object(forKey: contextSharedKey) as? Data,
           let shareData = try? JSONDecoder().decode(ShareData.self, from: data) {
            return shareData
        } else {
            return nil
        }
    }
}

ShareViewController的338,339行為進行圖檔壓縮。從外部分享的照片可能因為檔案過大造成儲存失敗，因此需先將圖片壓縮至目標寬度，再用二分壓縮法將圖片壓縮至目標大小再儲存。

extension UIImage{

	/// 壓縮圖片大小至目標寬度
    func compressImage(targetWidth: CGFloat) -> UIImage? {
        let width = self.size.width
        let height = self.size.height
        let scale = width / height
        let changedWidth: CGFloat
        let changedheight: CGFloat
        var sizeChange: CGSize
        
        if width <= targetWidth && height <= targetWidth {
            // 圖片寬或者高均小於或等於targetWidth時圖片尺寸保持不變，不改變圖片大小
            return self
        } else {
            if scale >= 1 {
                changedWidth = targetWidth
                changedheight = changedWidth / scale
                sizeChange = CGSize(width: changedWidth, height: changedheight)
            } else {
                changedheight = targetWidth
                changedWidth = changedheight * scale
                sizeChange = CGSize(width: changedWidth, height: changedheight)
            }
        }
        UIGraphicsBeginImageContext(sizeChange)
        self.draw(in: CGRect(x: 0, y: 0, width: sizeChange.width, height: sizeChange.height))
        let resizedImg = UIGraphicsGetImageFromCurrentImageContext()
        UIGraphicsEndImageContext()
        return resizedImg
    }
    
    /// 二分壓縮法
    func compressImageMid(expectedKb: Int) -> Data? {
        // 目標限制大小
        let limitSize = expectedKb * 1024 // 轉換成 byte
        // 壓縮比例
        var compression: CGFloat = 1
        guard var data = self.jpegData(compressionQuality: 1) else { return nil }
        Logger.log(message: "壓縮前kb: \(Double(data.count/1024)) KB")
        // 若原本尺寸就已經小於目標限制，則返回
        if data.count < limitSize {
            return data
        }
        // 循環壓縮
        var max: CGFloat = 1
        var min: CGFloat = 0
        for _ in 0 ..< 6 {
            compression = (max + min) / 2
            guard let compressData = self.jpegData(compressionQuality: compression) else { return nil }
            data = compressData
            Logger.log(message: "壓縮後kb: \(Double(data.count/1024)) KB")
            if CGFloat(data.count) < CGFloat(limitSize) * 0.9 {
                min = compression
            } else if data.count > limitSize {
                max = compression
            } else {
                break
            }
        }
        return data
    }
}
